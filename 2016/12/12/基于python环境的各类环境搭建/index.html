5. celery跑起来

  • 独立celery+python脚本
  • django-celery替代celery，在django项目中的使用方法
  • 监控界面：flower
  1. 安装：

pip install flower

  1. 使用方法：

运行服务打开http://localhost:5555：

flower --port=5555

或者从celery运行：

celery flower --address=127.0.0.1 --port=5555

Broker URL 和其他配置选项能够通过一个标准的celery选项：

celery flower --broker=amqp://guest:guest@localhost:5672//
